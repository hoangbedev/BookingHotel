# 🏨 Website Đặt Phòng Khách Sạn

## 📌 Giới thiệu
Dự án xây dựng một website đặt phòng khách sạn trực tuyến giúp người dùng tìm kiếm, lựa chọn và đặt phòng một cách nhanh chóng và tiện lợi.

Hệ thống hỗ trợ:
- Tìm kiếm khách sạn và phòng
- Xem thông tin chi tiết (giá, hình ảnh, mô tả)
- Đặt phòng trực tuyến
- Quản lý đơn đặt phòng
- Thống kê cho quản trị viên
